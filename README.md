# Escopo e objetivo

Projeto de cronometro do "Curso Vue Js do zero ao avançado na prática" (Udemy)

O projeto inclui:
(a) VueJs
(b) Yarn

## Iniciando...

- `git clone https://github.com/thiagostudier/chronometer-app.git`
- `cd chronometer-app`

> _Parte da configuração: Necessário instalar o yarn

- `yarn install`
- `yarn serve`
